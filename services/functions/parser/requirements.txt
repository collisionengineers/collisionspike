# Runtime dependencies for the Collision Engineers parser Function (Linux / FC1).
#
# Installed by the FC1 remote Oryx build (`--build-remote true`) on the Linux
# worker. The engine itself is VENDORED into this package as
# ./cedocumentmapper_v2/ (a top-level importable package) — it is NOT pip
# installed, so cedocumentmapper-v2 deliberately does NOT appear below.
#
# GUI / Windows-only deps from the sibling's pyproject are EXCLUDED here because
# they are never reached on the Function runtime path (DocumentMapperService ->
# readers -> rules -> normalisers -> record_to_dict). The vendored copy also
# drops ui/host.py + cli.py, the only modules that import them:
#   * tkinterdnd2 / pywebview  — desktop UI only (Tkinter / pywebview window)
#   * pywin32 (pythoncom/win32com) — Windows-only .doc COM path, lazily imported
#     and guarded; .doc still reads via the olefile/text-scrape fallback
#   * pydantic — declared in the sibling pyproject but NOT imported anywhere in
#     the engine source; omitted to keep the package lean.
