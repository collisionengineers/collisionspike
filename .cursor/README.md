# Cursor adapter

Canonical repository guidance lives in `AGENTS.md`; canonical role and skill
definitions live under `.agents/`.

- `agents/` and `skills/` are generated by
  `scripts/maintenance/generate-agent-adapters.mjs`.
- `rules/` contains small Cursor-specific routing rules.
- `hooks.json` and `hooks/` adapt the shared repository safeguards.

Regenerate adapters after changing `.agents` and verify parity with:

```text
node scripts/maintenance/generate-agent-adapters.mjs
node scripts/maintenance/generate-agent-adapters.mjs --check
```
