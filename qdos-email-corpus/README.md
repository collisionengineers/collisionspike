# QDOS email corpus

This directory combines the discovered QDOS email and case-document sets for internal evaluation.
The source trees remain separate under `sources/`; files have not been flattened or renamed.

`manifest.json` records every logical occurrence with its source group, original relative path,
owner, media type, byte size, and SHA-256 digest. Treat the copied source bytes as immutable.

The corpus contains real correspondence and case material. Keep it within the approved repository
workflow and do not publish or send it to an unapproved service.
